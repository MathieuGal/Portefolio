# Portfolio BTS SIO SLAM

Ce projet est un template de portfolio conçu pour les étudiants en BTS SIO option SLAM. Il est optimisé pour être facile à modifier et respecter les exigences des épreuves E4 et E6.

## Comment l'utiliser ?

1.  **Ouvrez le site** : Double-cliquez simplement sur le fichier `index.html` pour voir le site dans votre navigateur.
2.  **Modifiez le contenu** : Ouvrez le fichier `js/data.js` avec un éditeur de texte (Bloc-notes, VS Code, etc.). C'est ici que vous changerez votre nom, vos projets, vos compétences, etc.
3.  **Ajoutez vos fichiers** :
    *   Mettez votre CV dans `assets/documents/cv.pdf`.
    *   Mettez les images de vos projets dans `assets/img/projets/`.
    *   Mettez votre tableau de synthèse E4 dans `assets/documents/tableau_synthese.pdf`.

## Guide Détaillé
Pour savoir exactement quoi écrire dans chaque section pour valider vos épreuves, consultez le fichier [GUIDE_REMPLISSAGE.md](GUIDE_REMPLISSAGE.md).

## Structure du projet
*   `index.html` et autres fichiers `.html` : Les pages du site (ne pas toucher sauf si vous connaissez le HTML).
*   `css/style.css` : Le design du site.
*   `js/data.js` : **VOTRE FICHIER DE CONTENU**.
*   `js/main.js` : Le code qui fait fonctionner le site.
*   `assets/` : Dossier pour vos images et documents.
